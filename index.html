Qco for iOS

A simple library for encoding QR code in the iOS.

What is Qco?

Qco is a simple Objective-C library for encode QR code. I have designed this library so that we could create a QR code easily. Qco is an acronym of "QR code Creator of Objective-C". Please call it "Q-co".

This library is wrapper of C++ program. And it is used part of open source code at "Psytec Inc.".In detail, please refer to the following site.

  • http://www.psytec.co.jp/freesoft/02/ (Japanese only.)

How to install.

  1. Download Qco and build the project.
  2. Add "QCQRCode.h" and "libQco.a" in Xcode.
  3. In [TARGETS] -> [Build Phase] -> [Link Binary With Library], add "libQco.a", "UIKit.framework", "Foundation.framework" and "CoreGraphics.framework".
  4. Rename implementation file as follow. Because this library is wrapped C++ source code.
 XXXX.m -> XXXX.mm

How to use.

Qco has been designed to be able to make QR code in three steps.

  1. Initialize.
  2. Set parameters. (Option)
  3. Encode.
#inclide "QCQRCode.h"
...
// Initialize
QCQRCode * encoder = [[QCQRCode alloc] init];
// Set parameters.
encoder.moduleSize = 10;
// Encode
UIImage * qrcode = [encoder encodeWithText:@"Hello Qco."];

In detail, please check header file (QRCRCode.h).
